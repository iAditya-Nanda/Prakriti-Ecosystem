# Prakriti Tech Stack

## 🔙 Backend & Core
- **Primary Language**: Python 3.10+
- **API Framework**: Flask (RESTful)
- **Database**: PostgreSQL 14+
- **ORM**: SQLAlchemy
- **Authentication**: JWT (PyJWT)
- **CORS Handling**: Flask-CORS

## ⛓️ Blockchain Engine
- **Implementation**: Custom Python Hashing Logic
- **Hashing Algorithm**: SHA-256
- **Consensus**: Proof-of-Work (PoW)
- **Data Format**: JSON-based Ledger
- **CLI Subsystem**: Python `cmd` / `argparse`

## 🤖 Artificial Intelligence
- **Platform**: Ollama (Edge Inference)
- **Vision Model**: `prakriti-vision` (based on LLaVA/CLIP)
- **Chat Model**: `prakriti-chat` (based on Mistral/Llama)
- **Deployment**: Local Flask HTTP Microservices

## 📱 Mobile App
- **Core**: React Native
- **Runtime**: Expo
- **Navigation**: React Navigation
- **Icons**: Lucide React Native
- **Camera**: Expo Camera/ImagePicker

## 📊 Admin Dashboard
- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 🛠️ DevOps & Tools
- **Environment**: Linux/Ubuntu
- **Version Control**: Git
- **Dependency Management**: `pip` (requirements.txt) / `npm`
- **Networking**: Localhost / Tunneling for Mobile Testing
